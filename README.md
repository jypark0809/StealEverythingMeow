# 이븐아이 게임톤 8팀 다훔칠꼬양
인간에게 버려진 고양이들의 유쾌한 반격이 시작된다.

루팅게임과 육성게임이 합쳐진 루팅 육성시뮬레이션 

인간의 물건을 훔쳐 고양이를 모으고 집을 키워보자
